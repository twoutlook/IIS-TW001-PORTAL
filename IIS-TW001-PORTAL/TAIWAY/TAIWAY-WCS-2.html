任务下达（WMS调用）

方法名称 搬运任务下达
方法 http://[IP]:[端口]/api/WMSWCS/MoveTaskAdd
请求类型 POST
说明 出库、库对库搬运任务下达
方法名称 类型
FromBody JSON对象说明
属性名称 类型 字符长度 允许为空 描述 备注
输入 JSON对象
taskNo string 32 否 任务号
palletNo string 32 否 托盘号
bussinessType string 2 否 任务类型 1:入库 2:出库 5：库对库
locationFrom string 7 否 起始库位（站口）
locationTo string 7 否 目标库位（站口） 指定出库的站口号
deliveryTime string 20 否 任务下达时间
输入 JSON对象
success bool 否 true：成功，false：失败 调用成功，允许进入
errCode string 8 是 异常码，参考异常码定义
errMsg string 256 是 错误说明，当success为false时，errCode有值
输入 实例对象 {"lineId":"4","taskNo":"11145","palletNo":"10005861","bussinessType":"2","locationFrom":"0402604","locationTo":"C132","deliveryTime":"2020-09-10 11:53:56"}
输出 实例对象 {"success":"true","errCode":"","errMsg":""}

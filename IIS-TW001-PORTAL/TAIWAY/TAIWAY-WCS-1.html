1、本文档中的接口是浩登WMS提供，WCS调用

2、交互方式为提供WebApi接口服务

方法名称 任务状态反馈（WCS调用）
方法 http://[IP]:[端口]/api/WMSWCS/TaskStatesUpdate
请求类型 POST
说明 任务状态反馈
方法名称 类型 FromBody JSON对象说明
属性名称 类型 字符长度 允许为空 描述 备注
输入 JSON对象 taskNo string 32 否 任务号
palletNo string 32 否 托盘号
state string 4 否 任务状态 11：入库任务AGV搬运结束
12：入库任务下达主机命令
13：主机命令正常完成
14：主机命令强制完成
15：主机命令强制取消
22：出库任务下达主机命令
23：主机命令正常完成
24：主机命令强制完成
25：主机命令强制取消
52：库对库任务下达主机命令
53：主机命令正常完成
54：主机命令强制完成
55：主机命令强制取消
9999：任务异常（如WMS所选库位号无法使用）
※注
errMsg string max 是 异常信息
输入 实例对象 {"taskNo":"11297","palletNo":"10006160","state":"12","errMsg":null}
输出 实例对象 {"success":true,"errCode":null,"errMsg":null}

※注 12,22,52 可以继续下命令
